Qual é a ideia desse site?

Inclementação de API

API escolhida foi a XIVAPI

Como de inicio eu criei algo baseado em um site do jogo Final Fantasy XIV chamado lodestone, tive que ir atrás de uma api relacionada

O que a Api escolhida pode fazer?

Muitas opções de função estão disponiveis para ser utilizdas como um sumário com itens do jogo, missões entre outras funcionalidades. Porém apenas implementei a pesquisa de personagem onde apenas com um nick ou parte dele (Ex: Hugonez) vc consegue encontrar o nick e ID desse personagem e com isso puxar várias informações do mesmo
